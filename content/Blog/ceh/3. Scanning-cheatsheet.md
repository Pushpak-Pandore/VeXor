## 🅰️ Host Discovery (Nmap)

### 1. ARP Ping Scan

```bash
nmap -sn -PR <target ip>
```

* `-sn` → Ping scan only (No port scanning)
* `-PR` → ARP Ping Scan (used in local network)



### 2. UDP Ping Scan

```bash
nmap -sn -PU <target ip>
```

* `-sn` → Ping scan only
* `-PU` → UDP Ping Scan



### 3. ICMP ECHO Ping Scan

```bash
nmap -sn -PE <target ip>
```

* `-PE` → ICMP Echo Ping



### 4. ICMP Ping Sweep

```bash
nmap -sn -PE 10.10.1.10-23
```

* Scan IP range
* Detect live hosts



### 5. ICMP Timestamp Ping

```bash
nmap -sn -PP <target ip>
```

* `-PP` → Timestamp Ping



### 6. ICMP Address Mask Ping

```bash
nmap -sn -PM <target ip>
```

* `-PM` → Mask Ping



### 7. TCP SYN Ping

```bash
nmap -sn -PS <target ip>
```

* SYN-ACK → Host is UP



### 8. TCP ACK Ping

```bash
nmap -sn -PA <target ip>
```

* RST → Host is UP



### 9. IP Protocol Ping

```bash
nmap -sn -PO <target ip>
```

* Any response → Host is UP



    ### 📊 Ping Types Summary

| Type  | Flag | Response       | Use             |
| ----- | ---- | -------------- | --------------- |
| Echo  | -PE  | Echo reply     | Basic ping      |
| Time  | -PP  | Timestamp      | Bypass filters  |
| Mask  | -PM  | Mask reply     | ICMP blocked    |
| SYN   | -PS  | SYN-ACK        | Firewall bypass |
| ACK   | -PA  | RST            | Firewall detect |
| UDP   | -PU  | UDP/ICMP reply | External scan   |
| ARP   | -PR  | ARP reply      | Best for LAN    |
| Proto | -PO  | Any response   | Advanced        |



## 🅱️ Port & Service Discovery

### 1. TCP Connect Scan

```bash
nmap -sT -v <target ip>
```

* Full connection scan



### 2. SYN (Stealth) Scan

```bash
nmap -sS -v <target ip>
```

* Half-open scan (stealth)



### 3. Xmas Scan

```bash
nmap -sX -v <target ip>
```

* FIN + URG + PUSH flags



### 4. Maimon Scan

```bash
nmap -sM -v <target ip>
```

* No response → Open/Filtered
* RST → Closed



### 5. ACK Scan

```bash
nmap -sA -v <target ip>
```

* Detect firewall rules



### 6. UDP Scan

```bash
nmap -sU -v <target ip>
```

* Detect services like DNS, SNMP



### 7. NULL Scan

```bash
nmap -sN -T4 -A <target ip>
```

| Setting  | Value      |
| -------- | ---------- |
| TCP Scan | NULL (-sN) |
| Timing   | -T4        |
| Advanced | -A         |



## ⚡ Advanced Scans

### 1. Idle Scan

```bash
nmap -sI -v <target ip>
```

### 2. SCTP INIT Scan

```bash
nmap -sY -v <target ip>
```

### 3. SCTP COOKIE ECHO

```bash
nmap -sZ -v <target ip>
```

* No response → OPEN
* ABORT → CLOSED



### 4. Service Version Detection

```bash
nmap -sV <target ip>
```

### 5. Aggressive Scan

```bash
nmap -A 10.10.1.*
```

Includes:

* `-O` → OS detection
* `-sV` → Service version
* `-sC` → Default scripts
* `--traceroute`



## 🅲 OS Discovery

```bash
nmap -A <target ip>
nmap -O <target ip>
nmap --script smb-os-discovery.nse <target ip>
```



## 🅳 Firewall / IDS Evasion

```bash
nmap -f <target ip>
nmap -g 80 <target ip>
nmap --mtu 8 <target ip>
nmap -D RND:10 <target ip>
nmap -sT -Pn --spoof-mac 0 <target ip>
```

* Fragment packets
* Use trusted ports
* Decoy IPs
* Hide MAC
